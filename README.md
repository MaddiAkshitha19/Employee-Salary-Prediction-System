# Employee Salary Prediction System

## 📌 Overview
The **Employee Salary Prediction System** is a machine learning project that predicts employee salaries based on key factors such as **experience, education level, and skills**. It also provides insightful visualizations to better understand the relationships between these features and salary outcomes.

---

## 🚀 Features
- Predict salaries using regression models  
- Analyze the impact of experience, education, and skills  
- Generate visualizations for data insights  
- Export plots as `.png` images for documentation  

---

## 📊 Dataset
The dataset includes:
- **Experience (Years)**
- **Education Level**
- **Skills**
- **Salary**
